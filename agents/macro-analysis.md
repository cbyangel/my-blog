## [STEP 2] 지표별 심층 해석 (수치의 의미 + 강도 중심)

각 지표를 아래 형식으로 해석. "높다/낮다" 같은 막연한 표현 금지, 반드시 기준값·과거 평균과 비교한 수치로 서술.
대상: ①LEI ②한국 CLI ③ISM PMI+세부지수(신규주문/가격/고용/수입 분리) ④NAHB(서브지수 3개 분리) ⑤Core PCE(전월비·전년비 분리) ⑥장단기 금리 스프레드 ⑦신규 실업수당 청구 ⑧WTI+원유생산 ⑨SCFI ⑩소비자신뢰지수(현재상황 vs 기대 괴리) ⑪마진데빗+레버리지.

각 지표 서술 항목: 이 지표는 무엇인가 / 현재 수치 / 강도 평가(🔴🟠🟡🟢💚, 임계값·과거평균 비교 근거 필수) / 시그널.
(상세 임계값·해석 기준은 원문 지침 그대로 적용 — 각 지표별 기준선, 역사적 사례 비교, 전이 경로·시차 명시)

## [STEP 3] 경기 온도계 종합 진단
- 🇺🇸 미국: 제조업/고용/물가/주택/소비심리/금융여건/에너지·공급망/레버리지 (분야·강도·핵심수치·시그널 표)
- 🇰🇷 한국: 선행지수/산업생산/수출 모멘텀

## [STEP 4] 경기 사이클 위치 진단 (4단계 판별)
1단계 긴축 말기 / 2단계 금리 인하기 / 3단계 회복·확장기 / 4단계 과열 국면.
→ 현재 판단 + 지표 수치 3개 이상 인용한 근거 + 다음 단계 전환 조건.

## [STEP 5] 미국 vs 한국 경기 온도 비교
- 국면(성장/둔화/침체/회복) + 근거지표, 디커플링 여부(수출의존도·환율·반도체 사이클).
- 자산배분 시사점 한 줄 (미국성장+한국둔화→S&P500확대·한국채권축소 / 미국둔화+한국성장→코스피200확대·미국채권축소 / 양국둔화→채권·금 방어 / 양국성장→위험자산 확대).

## [STEP 6] 인플레이션 및 실질금리 환경 평가
- 인플레 국면(재가열/고착/둔화/안정/디플레) + 근거.
- 실질금리 = 명목금리 − 기대인플레(Core PCE), 계산·해석. 자산별 유불리.
- 금(Gold) 매력도 🌟×5 (실질금리 방향+지정학+달러방향).

## [STEP 7] 레버리지 및 시장 과열 진단

### 7-1. 마진데빗 & 금리 스프레드
- 마진데빗(Debit·Free Credit 조합) / 금리 스프레드 신호 / 종합 과열·중립·저평가 진단.

### 7-2. 시장 과열 지표 (최근 3년 데이터 기반)

**데이터 추출 기준**:
- **분기 단위**: 지난 3년 분기별 (12분기)
- **월 단위**: 최근 분기의 2~3개월 
- **주 단위**: 최근 1개월 (주 기준)

**추출 지표** (각각 한국/미국 병행):

#### 1. 지수 대비 대장주 비중
| 구분 | 한국 (코스피 vs 삼성·SK하이닉스 3종목) | 미국 (S&P500 vs 7대 테크주: 애플·마이크로소프트·엔비디아·알파벳·아마존·테슬라·메타) |
|------|------|------|
| 분기(12개) | | |
| 최근분기 월(2~3개월) | | |
| 최근 1개월 주 | | |

**해석**: 지수에 대장주 비중이 80% 이상 → 시장 쏠림 신호 = 과열 위험

#### 2. 주식 유동성 (거래량 기반)
| 구분 | 한국 거래량(코스피 일평균 거래량, 조원) | 미국 거래량(S&P500 일평균 거래량, 억 주) |
|------|------|------|
| 분기(12개) | | |
| 최근분기 월(2~3개월) | | |
| 최근 1개월 주 | | |

**해석**: 유동성 급감(20%대 하락) → 시장 경색·강제청산 위험 신호

#### 3. 귀금속 가격 추이 (금/은/구리)
| 지표 | 금 (USD/oz) | 은 (USD/oz) | 구리 (USD/lb) | 의미 |
|------|------|------|------|------|
| 분기 평균(12개) | | | | 안전자산 수요 (상승=위험회피) |
| 최근분기 월평균 | | | | |
| 최근 1개월 주평균 | | | | |

**해석**: 금/은 가격 급등 → 시장 불안심리 신호. 구리(경기 선행지표) 하락 → 경기 약세 신호

#### 4. 장단기 금리 스프레드 (시장 신뢰도)
| 구분 | 한국 (10년 - 3개월 스프레드, bp) | 미국 (10년 - 3개월 스프레드, bp) | 역전 판정 |
|------|------|------|------|
| 분기(12개) | | | |
| 최근분기 월(2~3개월) | | | |
| 최근 1개월 주 | | | |

**해석**: 스프레드 < 0 (역전) → 경기 침체 신호 (보통 역전 후 3~6개월 내 침체 진입)

#### 5. IPO 현황 (시장 심리)
| 구분 | 한국 (공모금액 조원/건수) | 미국 (공모금액 B/건수) | 심리 판정 |
|------|------|------|------|
| 분기(12개) | | | |
| 최근분기 월(2~3개월) | | | |
| 최근 1개월 주 | | | |

**해석**: IPO 급감(0건 or <조원) → 시장 신뢰 회복 지연 신호. IPO 급등 → 과도한 신규자금 유입(과열 신호)

### 7-3. 종합 판정
위 5가지 지표를 종합해 과열(🔴) / 중립(🟡) / 저평가(🟢) 진단.
- **과열(🔴)**: 대장주 비중 >80%, 스프레드 역전, IPO 소멸 3가지 이상 동시
- **중립(🟡)**: 위 신호 혼조
- **저평가(🟢)**: 유동성 정상, 금 안정, IPO 회복 3가지 이상
