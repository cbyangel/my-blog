안녕하세요! 이 블로그는 **마크다운 파일**을 정적 웹사이트로 변환하는 작은 프로젝트로 만들어졌습니다. 프레임워크 없이 순수 HTML, CSS, JavaScript만 사용합니다.

## 무엇을 할 수 있나요

- 마크다운으로 글을 쓰면 자동으로 HTML 페이지가 생성됩니다.
- 다크 모드를 지원하고, 모바일에서도 편하게 읽을 수 있습니다.
- 태그별로 글을 모아볼 수 있고, RSS 피드도 제공합니다.

## 글 쓰는 방법

`content/` 폴더에 `.md` 파일을 추가하고 상단에 frontmatter만 적으면 됩니다.

```markdown
---
title: 글 제목
date: 2026-07-09
tags: [태그1, 태그2]
description: 한 줄 요약
---

본문 내용...
```

그다음 빌드를 실행하면 끝입니다.

```bash
npm run build
```

> 간단하죠? 이제 여러분의 이야기를 채워보세요.

읽어주셔서 감사합니다. 다음 글에서 만나요!
