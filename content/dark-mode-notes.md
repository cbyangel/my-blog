이 블로그의 다크 모드는 세 가지 원칙으로 동작합니다.

## 1. 시스템 설정을 기본값으로

방문자가 운영체제에서 다크 모드를 쓰고 있다면, 별도 조작 없이도 다크 테마로 보입니다. `prefers-color-scheme` 미디어 쿼리를 사용합니다.

```css
@media (prefers-color-scheme: dark) {
  :root:not([data-theme="light"]) {
    --color-bg: #0f1115;
    --color-text: #e6e9ee;
  }
}
```

## 2. 수동 토글이 시스템 설정을 이긴다

헤더의 버튼을 누르면 `<html>`의 `data-theme` 속성이 바뀌고, 이 값이 시스템 설정보다 우선합니다.

## 3. 선택을 기억한다

선택한 테마는 `localStorage`에 저장되어 다음 방문 때도 유지됩니다. 페이지가 그려지기 전에 `<head>`의 인라인 스크립트가 이 값을 먼저 반영하므로 **화면 깜빡임(FOUC)** 이 없습니다.

> 색상은 모두 CSS 변수로 관리되기 때문에, 값 하나만 바꿔도 사이트 전체 테마가 바뀝니다.

간단하지만 사용성 면에서 중요한 디테일들입니다.
